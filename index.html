The purpose of this application is to easily allow dirt-simple dynamic plotting of functions. The equations may refer to any member of the standard Math object.

Per now, the application only supports basic plotting from one independant variable (x) and one time variable (t) in a cartesian coordinate system. Suggestions for extensions are most welcome.
